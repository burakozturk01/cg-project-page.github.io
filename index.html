Content Guard

Bilkent CS 2023-2024 Senior Design Project

The project we aim to create this year is a Twitter content tracker. Our primary purpose is to program a content tracker that helps the user monitor the time spent on a social media platform with an extensive range of content like Twitter.

Due to this extensive range, many people spend a lot of time on this kind of platform and need help to control this spent time. Thus, this time can become useless time spent on the flow of the social media platform on the subjects that users did not wish to focus on or research on this platform.

To create a solution to this problem, we conceived an idea of a content tracker plugin for Twitter that tracks the users’ browsing habits by category and offers features like filtering the content they choose not to invest their time in while browsing, listing the browsing habits of the user, enabling user to set the maximum time they would like to spend on the platform and restrictions based on categories and keywords.

For example, like the commonly used Chrome extension AdBlock’s feature of blocking the ads by removing them from the web page, our program will remove the tweets that the user does not wish to see according to the data our program collects and the preferences of categories they want to not to know the user makes that. Our program aims to enhance the user’s overall online experience and time management by offering a way to control the content they see on the Twitter platform.
